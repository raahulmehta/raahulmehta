## 👋 About Me

Full-Stack Developer who builds **immersive digital products** — not just functional apps, but experiences that feel alive. I specialise in the intersection of robust engineering and creative frontend work: real-time systems, 3D web, scroll-driven storytelling, and production-grade deployments.

**My stack runs deep:** From pixel-perfect React UIs with GSAP + Three.js animations, through Node.js/MongoDB backends with Socket.io real-time layers, all the way to containerised Docker deployments on VPS with Nginx/Caddy, CI/CD pipelines via GitHub Actions, and Kubernetes orchestration.

```
Frontend  →  React · Next.js · React Native · GSAP · Three.js · Framer Motion
Backend   →  Node.js · Express · Hono · NestJS · Fastify · REST · WebSockets
Database  →  MongoDB · Redis
DevOps    →  Docker · Kubernetes · GitHub Actions · Nginx · Caddy · AWS · VPS
Edge      →  Cloudflare Workers · Vercel · AWS Lambda
```

- 🔭 Currently working on enterprise marketplace apps and real-time analytics systems
- 🌱 Deep diving into **Kubernetes**, **Caddy**, and **advanced CI/CD patterns**
- 🎯 Speciality: animations that make people say *"wait, how did they do that?"*
- 💼 Available for freelance · [Upwork Profile](https://www.upwork.com/freelancers/raahulmehta)
